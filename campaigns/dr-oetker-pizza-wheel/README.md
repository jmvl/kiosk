# Dr. Oetker Pizza Wheel Campaign Package

Draft campaign package scaffold for the Belgian bilingual pizza wheel activation.

## Status

Prototype-safe scaffold. No real Dr. Oetker logos, product photos, bitmap art, or other trademarked brand assets are included in this package. Placeholder files exist only to exercise package validation and runtime wiring until approved assets are delivered with documented rights.

## Included

- `manifest.json` — package schema manifest with bilingual quiz, outcomes, ticket templates, placeholder assets, and visual wheel mapping.
- `campaign.json` — editable campaign copy/config model for admin/runtime previews.
- `module/index.html` — minimal presentation-only module scaffold that talks through the player bridge.
- `ticket-template/` — localized ticket examples with QR and human-readable ticket code placeholders.
- `assets/` — text placeholders for future approved brand/wheel/ticket bitmap assets.
- `legal/` — placeholder terms and privacy notes for pilot validation.
