# Chocomel Campaign Package

First campaign package for the retail kiosk activation platform.

## Status

Draft prototype foundation.

## Campaign Model

This package is intended to be loaded by the shared kiosk player and enforced by the local backend.

Current files:

- `campaign.json` — campaign manifest, copy, theme, wheel segments, questions, prizes, and reference assets

## Asset Rights Note

The current asset URLs point to public FrieslandCampina page imagery and are kept as prototype/reference assets only. Production usage requires brand/client approval and approved asset delivery.
